# UML editor

## Staging
To see the staging version: https://matehickey.github.io/uml-editor/  

## Getting Started

1.  Clone the repo:

        git clone https://github.com/matEhickey/uml-editor

2.  Install dependencies:

        yarn

    or

        npm install

3.  Run webpack:

        npm start
